# Design System

Auto-read when building UI.

## Philosophy

Channel: Vignelli, Rams, Stripe, Linear, Vercel
- Less but better
- Every element earns its place
- Typography-driven hierarchy
- Generous whitespace

## Typography

### Recommended (Free)
| Style | Font |
|-------|------|

| Modern | Satoshi |

## Type Scale

**Apps (tight):** 11, 12, 13, 14, 16, 18, 22, 28
**Marketing:** 12, 14, 16, 18, 20, 24, 30, 36

## Spacing

4px base: 4, 8, 12, 16, 24, 32, 48, 64, 96
When in doubt, add more space.

## Color

**Monochrome (Linear):** slate scale + single accent
**Warm (Stripe):** warm grays + gradient accent
**High contrast (Mercury):** pure black/white + minimal accent

## Radius

Pick ONE and use everywhere: sharp (0-2px), subtle (4-6px), or soft (8-12px)

## Checklist

Before shipping UI:
- [ ] Would someone know this is shadcn? (customize it)
- [ ] Typography is distinctive
- [ ] Consistent spacing (4px scale)
- [ ] Clear hierarchy (squint test)
- [ ] Generous whitespace
